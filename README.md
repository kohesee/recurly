# Recurly: Subscription Management App

A small subscription management app built while learning React Native and Expo. The project focuses on building a polished mobile UI, wiring navigation with Expo Router, and adding authentication with Clerk.

## What I Built

- Subscription management and tracking UI
- Tab navigation with Expo Router
- Clerk authentication setup
- Native Android development build flow
- EAS-ready Expo project structure

## Tech Stack

- React Native
- Expo
- Expo Router
- Clerk
- NativeWind
- EAS

## Run

Install dependencies:

```bash
bun install
```

Start the Expo dev server:

```bash
bunx expo start
```

For Clerk native support, use a development build instead of Expo Go:

```bash
bunx expo run:android
```
